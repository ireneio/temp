# meepShop lerna
The repo combined store, admin & mepp-ui.<br>

## How to use
### Development
```bash
$ npm run dev
```
### Build
```js
$ npm run prod
```
Build with Bundle Analyzer
```bash
$ BUNDLE_ANALYZE=both npm run prod
$ BUNDLE_ANALYZE=server npm run prod
$ BUNDLE_ANALYZE=browser npm run prod
```
